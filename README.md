# weather-app
A responsive and visually appealing weather application that allows users to search for real-time weather information for any city using the OpenWeatherMap API. Built with HTML, CSS, and JavaScript, this app provides an intuitive interface to display temperature, humidity, wind speed, and weather icons.

Features
City Search: Enter any city name to fetch current weather data.
Real-Time Data: Displays temperature (in Celsius), humidity, wind speed, and weather icons from OpenWeatherMap.
Responsive Design: Adapts seamlessly to different screen sizes, including mobile devices.
Error Handling: Alerts users for invalid city names or API errors.
Attractive UI: Features a gradient background, card-based layout, and smooth animations for an engaging user experience.
Technologies Used
HTML: Structure of the web application.
CSS: Styling, responsiveness, and visual effects.
JavaScript: Logic for API integration, event handling, and DOM manipulation.
OpenWeatherMap API: Source for real-time weather data.
Setup Instructions
Clone or Download the Repository:

Download the project files or clone the repository to your local machine.
Obtain an API Key:

Sign up for a free account at OpenWeatherMap.
Generate an API key from your dashboard.
Configure the API Key:

Open script.js.
Replace 'YOUR_API_KEY_HERE' with your actual OpenWeatherMap API key.
Run the Application:

Open index.html in any modern web browser (e.g., Chrome, Firefox).
Enter a city name in the search bar and click "Search" to view the weather data.
Optional: Host Locally:
If you have a local server (e.g., via Node.js or Python), serve the files to avoid CORS issues in some browsers.
